# Square Root - Javascript Example

This is just a super simple demonstration of how the square root algorithm works.

## Installation:

```
yarn install
```

## Usage:

```
import { squareRoot } from './path/to/squareRoot.js';
squareRoot(4); // 2
squareRoot(6); // 3
squareRoot(16); // 4
squareRoot(-1); // NaN
```
